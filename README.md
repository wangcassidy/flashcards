# Flashcards

A Shiny app for creating and using flash cards.

### How to run this app
In the R Studio console, run the code below:

```r
library(shiny)

runGitHub("flashcards", "wangcassidy", ref = "main")
```

Or, try it here: https://cassidywang.shinyapps.io/flashcard/

### Dependencies
- R Shiny
- Tidyverse
- ggfittext
